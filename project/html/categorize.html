Project Report

Given an image frame, find the most similar image frame from a database of images, using color image histograms

  • The number of bins for the histogram has been chosen to be 256/16 = 16 for each channel.
  • Thus the total number of bins are 16*16*16 = 4096.
  • The image has been blurred prior to histogram computation using an average filter.
  • For centering 80% of the smaller dimension is considered as the diameter of the circle

Remarks:

  1. We can see that the three provides similar result with minor differences
  2. The best result is given after centering refining, CCV and Color histogram gives the least expected result in most cases
  3. In some cases like 1,9,12 etc we get very good result
  4. In cases like 17,18,24 we get some of the very unexpected results
